# Homies

A browser idle clicker built with Vite, React, TypeScript, Phaser 3, and Tailwind.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Vercel

Use the included `vercel.json` and add the custom domain `homies.hodlhq.app` in your Vercel project settings.
